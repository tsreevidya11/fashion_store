# 🛍️ Fashion Store E-Commerce Web Application

## 📌 Overview

Fashion Store is a Java Full Stack E-Commerce Web Application developed using JSP, Servlets, JDBC, MySQL, HTML, and CSS following the MVC Architecture.

The application allows users to browse products, search items, filter by category, add products to cart, place orders, and view order history.

---

## 🚀 Features

### User Module

* User Registration
* User Login & Logout
* Session Management

### Product Module

* Product Listing
* Product Search
* Category Filtering
* Product Details View
* Product Size Selection

### Shopping Module

* Add to Cart
* Remove from Cart
* Cart Management
* Checkout Process

### Order Module

* Place Orders
* View Order History
* Order Details
